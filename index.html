Weather

A command line weather client.

View on GitHub Download .zip Download .tar.gz

This application displays weather data for a given location and timeframe.

Basic usage

$ ./w.py houston today

* Houston, United States
*
* Thu 14 May 2015
* Mostly Cloudy, with a high of 29C and a low of 23C
